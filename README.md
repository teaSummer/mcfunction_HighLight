# mcfunction 高亮
MC指令文件高亮增强 (MT管理器)

（准确而言，这是MT的语法文件）


## 如何进行高亮？

### 下载

从仓库上下载 `.mtsx` 文件，

或通过 `123云盘` 链接进行下载：

https://www.123pan.com/s/vhjA-ndYsH.html

### 使用

打开 `MT管理器`，找到下载好的文件，

单击这个文件，然后选择 `安装`！

然后就可以打开 `.mcfunction` 文件啦！

这将会让它变得丰富多彩！
